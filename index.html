Memahami Box Model dan Styling Modern

Ditulis pada Juni 2026

Apa itu CSS Box Model?

Box Model adalah fondasi tata letak di web. Setiap elemen HTML pada dasarnya adalah sebuah "kotak" yang terdiri dari:

  • Content: Isi elemen (teks/gambar).
  • Padding: Ruang kosong transparan di dalam kotak, di sekeliling konten.
  • Border: Garis batas kotak.
  • Margin: Ruang kosong transparan di luar kotak, memisahkan elemen satu dengan lainnya.

Penjelasan Styling Tipografi

Properti font-family digunakan untuk mengubah jenis huruf agar terlihat lebih profesional (seperti sans-serif). Sementara itu, background-color dan color membantu mengatur warna latar dan warna teks agar kontras dan mudah dibaca.
